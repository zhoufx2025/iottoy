# 1.自动抓取小车（2025.4.30）
## 大纲（2025.4.30）
  利用树莓派当作控制中枢，同时安装了homeassistant，上接互动设备，下接驱动设备。
  ### 系统架构图
  ```mermaid
  graph TD
      A[语音控制] --> B[智能中枢]
      B[智能中枢] --> D[树莓派 5]
      C[手柄控制] --> D
      D <--> E[相机]
      D <-->|I2C 通讯| F[驱动模组]
      N[7.4V 锂电池] -->|供电| F
      F -->|供电| D
      F --> G[4 路电机]
      F --> H[8 路舵机]
      G --> I[4 轮差速驱动]
      H --> J[2 自由度云台]
      H --> K[6 自由度机械手]
  ```
